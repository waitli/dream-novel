## 📋 项目概述

这是一个基于 **雪花写作法** 的 AI 小说创作工具，支持智能生成小说架构、角色体系、世界观和章节内容。采用 Vue 3 + Vite + TailwindCSS + Naive UI 技术栈，支持深色模式、本地存储、多项目管理。

### 核心功能
- 📖 雪花写作法流程（核心种子 → 角色动力学 → 世界观 → 情节架构）
- 🎭 角色弧光理论（驱动力三角、关系冲突网）
- 📊 悬念节奏曲线设计
- ✍️ 章节正文生成（支持流式输出）
- 🧭 灵感罗盘（人物关系图谱可视化）
- 📤 多格式导出（TXT/Markdown）

### API 层

#### `src/api/llm.js`
**作用**: LLM API 服务封装
- **`chatCompletion(config, prompt, onStream)`**:
  - 支持流式/非流式响应
  - 流式：使用 `fetch` + `ReadableStream` 逐块解析
  - 非流式：使用 `axios` 请求
- **`cleanResponse(text)`**:
  - 清理 AI 响应（移除 markdown 代码块）

#### `src/api/generator.js`
**作用**: 小说生成服务（核心业务逻辑）
- **`generateArchitecture(project, apiConfig, onProgress)`**:
  - 5 步流程：核心种子 → 角色体系 → 角色状态 → 世界观 → 情节架构
  - 每步调用对应的 prompt 函数
- **`generateChapterBlueprint(project, apiConfig, onProgress)`**:
  - 支持一次性生成或分块生成（根据 maxTokens 计算 chunkSize）
  - 解析已有章节，支持增量生成
- **`parseChapterBlueprint(blueprint)`**:
  - 解析大纲文本为结构化数组（章节号、标题、定位、作用等）
- **`generateChapterDraft(project, chapterNumber, apiConfig, onProgress)`**:
  - 区分第一章和后续章节（使用不同 prompt）
  - 包含前章结尾段、前文摘要、角色状态等上下文
- **`finalizeChapter(project, chapterNumber, chapterText, apiConfig, onProgress)`**:
  - 更新前文摘要和角色状态
- **`enrichChapter(chapterText, wordNumber, apiConfig, onProgress)`**:
  - 扩写章节内容
- **`exportNovelToText/Markdown(project)`**:
  - 导出小说为 TXT 或 Markdown 格式

#### `src/api/compass-generator.js`
**作用**: 灵感罗盘图谱生成服务
- **`generateBaseGraph(project, apiConfig, onProgress)`**:
  - 从架构数据提取基线图谱（节点 + 边）
  - 输出格式：`{ nodes, edges }`
- **`generateChapterSnapshots(project, apiConfig, onProgress)`**:
  - 从已写章节批量生成快照（增量更新）
  - 使用 delta 格式（newNodes, updatedNodes, removedNodeIds 等）
- **`auditCompassGraph(project, apiConfig, onProgress)`**:
  - 逻辑审计（检测 dead_reappear, missing_setup 等逻辑错误）
- **`generateChapterGraph(project, chapterNum, chapterText, apiConfig, onProgress)`**:
  - 从单章内容提取独立关系图谱

## 🔄 核心业务流程

### 1. 小说架构生成流程
```
用户创建项目
    ↓
点击"生成架构"
    ↓
generateArchitecture()
    ├─ Step 1: coreSeed (核心种子)
    ├─ Step 2: characterDynamics (角色体系)
    ├─ Step 2.5: characterState (角色状态)
    ├─ Step 3: worldBuilding (世界观)
    └─ Step 4: plotArchitecture (情节架构)
    ↓
保存到 project
```

### 2. 章节大纲生成流程
```
点击"生成大纲"
    ↓
generateChapterBlueprint()
    ├─ 构建 novelArchitecture 文本
    ├─ 计算 chunkSize（根据 maxTokens）
    ├─ 解析已有章节
    └─ 分块生成或一次性生成
    ↓
parseChapterBlueprint() → 结构化章节数组
    ↓
保存到 project.chapterBlueprint
```

### 3. 章节正文生成流程
```
选择章节 → 点击"生成草稿"
    ↓
generateChapterDraft()
    ├─ 第 1 章：使用 firstDraft prompt
    └─ 后续章节：使用 nextDraft prompt（包含前文摘要、前章结尾）
    ↓
显示草稿 → 用户可编辑
    ↓
点击"保存并定稿"
    ↓
finalizeChapter()
    ├─ 更新 globalSummary
    └─ 更新 characterState
    ↓
generateChapterGraph() → 本章关系图谱
    ↓
自动跳转到下一章
```

### 4. 灵感罗盘图谱生成流程
```
点击"生成图谱"
    ↓
generateBaseGraph()
    ├─ 调用 extractGraph prompt
    ├─ 解析 JSON 响应
    └─ 生成基线快照 (key: "0")
    ↓
保存到 project.graphData

有章节后 → 点击"更新章节快照"
    ↓
generateChapterSnapshots()
    ├─ 遍历未分析的章节
    ├─ 调用 extractChapterDelta prompt
    ├─ 应用 delta 到前一快照
    └─ 生成新快照 (key: 章节号)
```
